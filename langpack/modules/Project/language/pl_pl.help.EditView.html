Projects

To create a project, enter the following information:

  • Name. Enter a name for the project.
  • Team. Enter the name of the team that is allowed to view the project; to select from the Team list, click Select and click the team name.
  • Assigned to. Enter the name of the user who has ownership of the project. By default it is assigned to you.
  • Description. Enter a brief description of the project.
To create the project, click Save; click Cancel to exit the page without creating the project.
